Introduction

KLAT OS is a modern open source operating system built with the Jowo Kernel. This documentation will guide you through the installation, configuration, and development of KLAT OS.

System Requirements

  • x86_64 or ARM64 processor
  • 512 MB RAM minimum (2 GB recommended)
  • 2 GB free storage space
  • Bootable USB or virtual machine

Installation

Before installing KLAT OS, ensure your system meets the minimum requirements and prepare a bootable USB drive.

Creating Bootable USB

# Download the ISO
wget https://klat.asia/downloads/klat-os-1.0.iso

# Create bootable USB (Linux)
sudo dd if=klat-os-1.0.iso of=/dev/sdX bs=4M status=progress

# Create bootable USB (Windows)
# Use Rufus or Balena Etcher

Quick Start

After installation, boot into KLAT OS and follow the setup wizard.

# Open terminal
# Update system
sudo klat-update

# Install apps
sudo klat-install browser
sudo klat-install office

# Check system info
klat-info

Jowo Kernel

The Jowo Kernel is the heart of KLAT OS. It is a modern microkernel designed for performance, security, and stability.

Kernel APIs

// Memory allocation
void* memory_alloc(size_t size);

// Process creation
pid_t process_create(void* entry, int argc, char** argv);

// IPC messaging
int ipc_send(pid_t dest, ipc_message_t* msg);

// Driver registration
int driver_register(const char* name, driver_methods_t* methods);

API Reference

KLAT OS provides a rich API for application development.

#include <klat/system.h>
#include <klat/io.h>
#include <klat/net.h>

int main() {
    // Initialize KLAT API
    klat_init();

    // Create a window
    window_t* win = window_create("My App", 800, 600);

    // Main loop
    while (window_is_open(win)) {
        event_t evt;
        while (window_poll_event(win, &evt)) {
            handle_event(&evt);
        }
        window_render(win);
    }

    window_destroy(win);
    return 0;
}

Building from Source

# Clone the repository
git clone https://github.com/KlatAsia/KLAT-OS.git
cd KLAT-OS

# Build the kernel
make kernel

# Build the desktop environment
make desktop

# Build applications
make apps

# Create bootable ISO
make iso

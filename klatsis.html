Klat SIS

Comprehensive School Information System built with Laravel 12 backend and React + Vite + TypeScript frontend. Manages students, teachers, academics, attendance, grades, CBT exams, e-learning, finances, and parent portal. RBAC multi-role system, offline mode, real-time dashboard, AI-powered analytics, and full Indonesian education support.

Key Features

Complete School Management

Multi-Role RBAC

JWT + 2FA authentication with role-based access control. Support for admin, principal, teacher, student, parent, and staff roles with granular permissions.

Attendance System

Manual entry, QR code scan, RFID card, and fingerprint attendance. Real-time tracking, geo-fencing, late/early detection, and automatic parent notifications.

Grade Management

Pengetahuan, Keterampilan, Sikap, Rapor, and Ranking. Automated report cards, class ranking, subject analysis, and GPA calculation per Indonesian curriculum standards.

CBT Online Exams

Question bank management, random question selection, exam timer, auto-correction, anti-cheat measures, and instant result analytics for Computer Based Testing.

e-Learning Platform

Learning materials, assignments, discussion forums, file submissions, grading, and progress tracking. Built-in LMS without needing separate platforms.

Finance & SPP

Tuition (SPP) management, invoice generation, payment tracking, installment plans, overdue reminders, and comprehensive financial reports for school administration.

Parent Portal

Dedicated parent view for grades, attendance, schedules, payment status, teacher communication, and chat. Keep parents involved in their child's education.

AI Analytics

AI-powered student performance analysis, dropout prediction, learning pattern insights, automated recommendations, and natural language chatbot for school data queries.

Notifications

WhatsApp, Email, and Telegram notifications for attendance, grades, payments, announcements, and emergency alerts. Parents and students stay informed automatically.

FAQ

Frequently Asked Questions

Is Klat SIS free?

Yes! Klat SIS is 100% free and open source (GPL-3.0). No subscription fees, no per-student fees, no hidden costs. All features are available including CBT, e-learning, and AI analytics.

How many students can it handle?

Klat SIS supports 100K+ student records with optimized queries, proper indexing, and Redis caching. Tested with single schools of 5,000+ students and district-wide deployments of 100,000+.

Does it support Kurikulum Merdeka?

Yes! Klat SIS fully supports Kurikulum Merdeka (Indonesia's current curriculum) including PPKn, assessment types (Pengetahuan, Keterampilan, Sikap), and P5 project tracking. Also supports Kurikulum 2013 for transition periods.

Can it work offline?

Yes! Klat SIS has an offline-first PWA architecture. Teachers and students can use the system without internet. Data syncs automatically when connection is restored. Perfect for areas with unreliable internet.

How does the CBT system work?

The CBT module includes a question bank, random question selection per exam, configurable timers, auto-grading for multiple choice, anti-cheat tab switching detection, and instant result analytics with detailed reports.

Can I migrate from another system?

Yes! Klat SIS includes CSV/Excel import tools for students, teachers, classes, and historical data. We provide migration guides and can assist with large-scale data imports from other SIS platforms.

KLAT OS

Sistem operasi modern berbasis JowoKernel — kernel monolitik 64-bit yang dikembangkan sepenuhnya dari nol. Dibangun untuk performa, stabilitas, dan kompatibilitas POSIX.

Download Documentation GitHub Forum

Filosofi

Pengembangan KlatOS didasarkan pada prinsip-prinsip utama yang memastikan kualitas di setiap lapisan sistem.

⚙

Modern Architecture

Seluruh subsistem menggunakan arsitektur modern berbasis object-oriented dan event-driven.

⚡

Performance First

Seluruh jalur kritis dirancang untuk meminimalkan overhead: zero-copy VMO, event-driven WaitQueue, non-blocking I/O.

📄

POSIX Compatibility

Mengimplementasikan antarmuka POSIX sehingga berbagai aplikasi dan pustaka dapat dipindahkan dengan perubahan seminimal mungkin.

🎨

Graphics Stack

Graphics stack modern berbasis DRM dengan dumb buffer, page flipping, VBlank synchronization, zero-copy framebuffer.

🔒

Security & Reliability

Per-process virtual address space, process isolation, guard pages, reference counting, automatic resource cleanup.

📦

Clean Architecture

Kernel tidak dibangun dari kumpulan patch. Desain modular dengan batas antarsubsistem yang jelas.

JowoKernel

Kernel monolitik 64-bit yang menyediakan seluruh layanan sistem operasi kepada userspace melalui antarmuka syscall yang stabil.

Process Management

Multi-process & multi-thread scheduler, context switching, process groups, sessions, job control, signal delivery.

Memory Management

Physical Memory Manager, VMM, kernel heap, slab allocator, VMO, memory mapping, shared memory, guard pages.

Filesystem

Virtual File System, TmpFS, ProcFS, DeviceFS, file descriptor table, symbolic links, hard links, permission management.

IPC

Pipes, Unix Domain Socket, shared memory, zero-copy buffer sharing, file descriptor passing (SCM_RIGHTS).

Event Infrastructure

WaitQueue, poll, select, epoll, event-driven wakeup, non-blocking file descriptors — menghilangkan polling periodik.

Desktop Platform

Display server, window manager, input server, desktop shell, launcher, notification service, session manager.

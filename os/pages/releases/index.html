KlatOS 1.0

Juli 2026 — Rilis Stabil Terbaru

Apa yang Baru

  • JowoKernel v1.0 — kernel monolitik 64-bit
  • Process Management: multi-process scheduler, signal delivery, job control
  • Memory Management: PMM, VMM, VMO, slab allocator, guard pages
  • Filesystem: VFS, TmpFS, ProcFS, DeviceFS, permission management
  • IPC: Unix Domain Socket, shared memory, SCM_RIGHTS, epoll
  • Event-driven kernel dengan WaitQueue
  • Graphics Stack: DRM, dumb buffer, page flipping, VBlank sync
  • libdrm compatibility layer
  • Desktop Platform: display server, window manager, shell, launcher
  • POSIX runtime dengan 30+ syscall
